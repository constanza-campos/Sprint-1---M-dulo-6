# 📋 KanbanPro - Gestión de Proyectos Fullstack

Aplicación integral para la gestión de tareas basada en metodología Kanban.

## 🛠️ Tecnologías Utilizadas
- **Backend:** Node.js & Express
- **Persistencia:** PostgreSQL con ORM Sequelize
- **Frontend:** HTML5, CSS3, Bootstrap 5

## 🚀 Instalación y Uso
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Configurar archivo `.env` con credenciales de PostgreSQL.
4. Crear base de datos `kanban_pro`.
5. Ejecutar `npm run db:seed` para poblar datos iniciales.
6. Iniciar con `npm run dev`.
